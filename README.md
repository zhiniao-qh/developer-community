## developer-community
简介：一个基于 SpringCloud 的开发者社区。

### 1、技术栈
- 后端：SpringCloud、MyBatis Plus、MySQL、Redis、MongoDB、Elasticsearch、RabbitMQ ...
- 前端：Vue.js、Element UI ...

### 2、模块划分

| module | 模块名 |
| :---: | :---: |
| dc-common | 公共模块 |
| dc-base | 基础模块 |
| dc-article | 文章模块 | 
| dc-gathering | 活动模块 | 
| dc-qa | 问答模块 |
| dc-recruit | 招聘模块 |
| dc-spit | 吐槽模块 | 
| dc-search | 搜索模块 |
